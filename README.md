# meridian-prototype
A prototype for a cross-domain reasoning system.
Live demo: [https://yourname.github.io/meridian-prototype](https://anoushka-j.github.io/meridian-prototype/)

Cross-domain reasoning interface. Select a source domain,
a target domain, describe a problem, and the system surfaces
structural transfers between fields — including where the
analogy breaks down.

Built with Claude code.
